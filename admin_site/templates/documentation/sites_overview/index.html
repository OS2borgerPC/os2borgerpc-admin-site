{% translate "Deleting a site" %}

{% blocktranslate trimmed %} To start the process of deleting a site, click on the dropdown symbol on the right side of the button, and then on 'Delete Site'. Note that if you have more than five computers associated with the site, the button will be inactive and deletion is not possible. You will need to manually remove computers for the deletion to be possible. When clicking on the 'Delete Site' button, you will be redirected to a page where you must again confirm the deletion to finalize it. {% endblocktranslate %}
